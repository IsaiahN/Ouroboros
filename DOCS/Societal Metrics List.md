Core Theory:

"If you want general intelligence, you must build a society. If you build a society with persistent memory and weighted integration, consciousness will emerge as a side effect."

Core Rules: Goodhart’s Law is navigated by treating metrics as disposable instruments, never as truths

1. No single metric should ever control survival, budget, or prestige.
2. Rotate metrics faster than agents can adapt (Metrics must be ephemeral, not institutionalized.)
3. Separate evaluation from optimization
-Metrics used to observe should not be the same as those used to reward
-Evaluation metrics should be partially hidden or lagged
-Optimization metrics should be noisy and incomplete
4. Use disagreement as a signal
Uniform agreement is suspicious.
Healthy systems track:
-variance
-dissent
-prediction spread
If all metrics agree perfectly, either:
-the problem is trivial
-or the system is already gaming them
Disagreement indicates:
-unexplored space
-robustness
-optionality
5. Reward robustness, not peak performance
Better targets:
-performance under perturbation
-recovery after failure
-consistency across contexts
6. Slack allows:
-mistakes without collapse
-exploration without punishment
-non-optimized behavior to persist
